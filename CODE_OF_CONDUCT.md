# Code of Conduct

We want this project to be welcoming, respectful, and easy to contribute to.

## Expected Behavior

- Be kind and constructive.
- Assume good intent.
- Give feedback on code, not on people.
- Respect different experience levels and viewpoints.

## Unacceptable Behavior

- Harassment, discrimination, or personal attacks
- Trolling, intimidation, or intentionally inflammatory behavior
- Sharing private information without permission
- Repeated disruption of conversations or reviews

## Reporting

If you experience or witness a problem, use the repository's private security or moderation channels when available. If no private channel is available yet, open a GitHub issue describing the problem and avoid public escalation in the thread.

## Enforcement

Maintainers may remove, edit, or close content that violates this code of conduct.
